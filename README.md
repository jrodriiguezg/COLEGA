# OpenKompai_nano
Asistente personal proactivo, especialmente útil en entornos domésticos o para personas que necesitan apoyo en su día a día. Su objetivo principal es facilitar la interacción con el usuario a través de comandos de voz y una interfaz gráfica sencilla

Se llama **Nano** porque, a diferencia de asistentes más complejos que utilizan grandes modelos de inteligencia artificial (IA) basados en aprendizaje profundo y procesamiento de lenguaje natural (NLP) avanzado, OpenKompai Nano opera con un sistema de instrucciones predefinidas. Esto significa que su "inteligencia" se basa en un conjunto de reglas y patrones (los intents.json) que asocian frases clave (triggers) con acciones específicas (action) y respuestas preescritas (responses). Este enfoque lo hace ligero, eficiente y adecuado para dispositivos con recursos limitados, como una Raspberry Pi, sin sacrificar la funcionalidad esencial para su propósit

La instalacion solo ha sido probada en Raspberry pi 4B 4Gb con Raspberry Pi OS Legacy Lite 64 Bits
(Debian 12 Bookworm)

```bash
curl -sSL https://raw.githubusercontent.com/jrodriiguezg/OpenKompai_nano/refs/heads/main/install.sh | bash 

```
